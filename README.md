# Reinforcement Learning Study (DQN / SARSA)

이 레포지토리는 강화학습을 학습하며  
기존 공개 코드를 기반으로 **코드 구조를 분석하고 주석을 추가하며 공부한 기록**입니다.

## 학습 목표
- DQN, SARSA 알고리즘 구조 이해
- replay buffer, target network 동작 분석
- 실제 코드 흐름을 따라가며 강화학습 구현 감각 익히기
